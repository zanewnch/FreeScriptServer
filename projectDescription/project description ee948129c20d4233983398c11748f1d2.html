project description

技術層面:

  • 前端

    我主要使用VUE3 進行開發 with ts, css 部分使用tailwind css, 額外組件大部分來自於element-plus

    組件部分的參數傳遞我全部使用pinia store 進行管理, 然後用axios with interceptor 來處理request

  • 後端

    我使用express.js with ts, 然後透過MVC當中的MVVM架構來進行設計，在後端我設計了router-controller-repo 三層來處理api design, 並使用class封裝的方式更好管理相關的function

  • 未登入提醒

    透過檢測登入cookie是否存在來判斷是否登入，未登入會跳出尚未未登入的提醒

  • backend cookie 檢查

    所有backend api 都有middleware, 需要攜帶cookie 才可以正常response

    現在因為尚未登入 所以未顯示任何資訊

  • 註冊頁面

    可以選擇直接key column value, 或是直接使用google進行註冊

  • 如果已經有註冊過的帳號，會顯示alert 說明帳號已經存在

    透過api response status 來判斷錯誤原因與內容

  • 註冊完成會直接到login頁面 一樣可以直接透過google 登入
